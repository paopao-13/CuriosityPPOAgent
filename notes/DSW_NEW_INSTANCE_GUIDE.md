# 阿里云 PAI-DSW 新建实例填写指南（资源包自动抵扣）

> ⚠️ **重要更正（2026-07-18）**：PAI-DSW **没有"资源包配"这个计费模式**。
> 正确机制：实例按「公共资源（按量付费）」创建；若账户有 DSW 资源抵扣包（CU·H），系统会**后台自动**抵扣按量费用，**无需、也无法在实例配置里手动选包**。
> 之前文档写的"计费方式选资源包配"是误读，已删除。你在创建/修改页"找不到资源包可抵扣"是正常的——那个下拉本来就不存在。
> 命令行见 `ATARI_RESUME.md`。界面措辞以控制台实际为准。

---

## 入口路径
1. 登录阿里云 → **PAI 控制台（机器学习平台 PAI）**
2. 左上角**地域**选成和你的资源包一致的地域（关键，见下方"创建前必做"）
3. 左侧 **交互式建模（DSW）** → **新建实例**

---

## 表单逐项填写表

| 字段 | 填什么 | 说明 / 坑 |
|------|--------|-----------|
| **工作空间** | 默认工作空间 | 没动过就选默认 |
| **实例名称** | `atari-curiosity-ppo` | 自定义，能认出即可 |
| **资源类型** ⚠️ | **公共资源**（=按量付费） | **不要找"资源包配"**——那不是 DSW 的选项。资源包抵扣是后台自动的。 |
| **资源规格** ⚠️ | 选**资源包支持抵扣清单内**的 A10 机型：**`ecs.gn7i-c8g1.2xlarge`（A10, 30GB）** | ⚠️ 你之前用的是 `ecs.gn7r-c8g1.2xlarge`，**gn7r 很可能不在抵扣范围**，这正是旧实例扣现金停机的元凶之一。务必选 **gn7i**（i 不是 r）。官方支持抵扣的 DSW 入门机型：ecs.gn7i-c8g1.2xlarge / ecs.gn6v-c8g1.2xlarge / ecs.g6.xlarge。 |
| **镜像** | 官方 **PyTorch GPU** 版（如 `pytorch:2.x-gpu-py3xx`） | 建好先验证 `torch.cuda.is_available()`，False 则按 `ATARI_RESUME.md` 重装 cu121 torch。 |
| **系统盘** | 默认 100 GiB 免费云盘即可（公共资源默认送） | ⚠️ 不要扩容——扩容部分停止实例后仍持续计费。 |
| **挂载数据集 / 网络** | 默认，不用挂 | 代码从 GitHub clone |

## 创建前必做：先去"资源包"页核实包能抵扣 DSW

> 这是诊断旧实例为何没被抵扣的关键一步。

- 阿里云控制台 → **费用中心 / 我的资源包** → 找那张 **100 CU·H** 的包
- 逐项确认：
  1. **适用产品** = 交互式建模（DSW）。若是 DLC / EAS 的包，DSW 费用无法用它抵扣 → 需重买 DSW 包。
  2. **地域** 与你要建实例的地域一致（不一致则不可见/不可抵扣）。
  3. **状态** = 可用、有余额（未用完/未过期）。
  4. **支持抵扣的规格清单**（包说明里列）→ 是否含 `ecs.gn7i-c8g1.2xlarge`。
- 把以上 4 项截图发我，我帮你确认到底是哪条卡住了。

---

## 老实例为何"改不了配置"
- 运行/停止的 DSW 实例**不能改计费方式**（官方明确"公共资源无法转为包年包月"），且资源包抵扣是后台自动的、不在实例配置里。
- 所以"修改老实例配置看不到资源包"是正常现象 → **直接新建一个规格正确的实例即可**，旧的欠费停着的可以删掉。

---

## 创建后
- 实例变 **「运行中」** 即按量计费，但**有 DSW 资源包会自动抵扣**，不再扣现金（前提是规格/产品/地域都对）。
- 进终端跑验证 + 训练（命令见 `ATARI_RESUME.md`）。

---

## 额度真实换算（重要，纠正旧文档乐观估计）
- A10 实例约消耗 **5–7 CU·H/小时**（不同包单价不同，以资源包页显示为准）。
- **100 CU·H ≈ 14–20 小时** A10 训练（旧文档写的"50–100h"严重高估，已作废）。
- 主线 10M 约 8–10h → 占一半多；消融要省着规划，不要一次性排满。
- 实例**停止**后不再消耗计算时（但扩容系统盘仍计费）。用完立刻点「停止」。

---

## 一句话避坑
**资源类型选「公共资源」+ 规格选 `ecs.gn7i-c8g1.2xlarge`（不是 gn7r）+ 先确认包的"产品=DSW/地域一致"= 费用自动从 100 CU·H 抵扣，不再扣现金。**
